j = 0,1,2 をそれぞれ a,b,c とする.

dp[i][j]: i日目にjを選んだときの幸福度の総和の最大値
ただし, dp[0][j] = 0 (j = 0,1,2)

また, i = 1, 2, ..., n について,
dp[i][0] = max(dp[i-1][1], dp[i-1][2]) + a[i-1]
dp[i][1] = max(dp[i-1][2], dp[i-1][0]) + b[i-1]
dp[i][2] = max(dp[i-1][0], dp[i-1][1]) + c[i-1]

このとき, max(dp[n][0], dp[n][1], dp[2]) が答えとなる.
